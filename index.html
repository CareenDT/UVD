UVD (или декларация диалогов)

Документация синтаксиса для написания диалогов[demo]

Основы

Каждая реплика должна начинаться с #

# блаблабла
# никита дурак
///
# Илья жирный

/// - нужен для ожидания нажатия кнопки продолжения

Если реплики написаны как выше, то она произноситься никем. (ну или нарратором|внутренним голосом и тд.)

Чтобы дать авторство персонажу, добавьте [c N] N=имя перса после #

# [c isha] что-то не остроумное
# что-то остроумное*
# [c eugene] что-то ещё более остроумное

*Каждую новую реплику пеосонаж сбрасывается

Контроль

Паузы

Для "паузы в речи" можно добавить [wait N] N=число !можно писать десятичные через точку (1.5)!

# Текст [wait 1.5] после паузы

Скорость

Чтобы изменить скорость ввода можно написать [spd N] N=символы в секунду

# Нормальный текст в 0.1 [wait 1] [spd 1] капец какой медленный текст

!Скорость сбрасывается в начале каждой реплики

Стилизация(ограниченно)

У нас есть типы диалог. окна[dialog box] (пока нет bruh) Стили: - Стилей нет но вы держитесь

Чтобы задать стиль dialog box-а нужно написать (ОБЯЗАТЕЛЬНО)ВНАЧАЛЕ после # [style N] N=название стиля из верхнего списка.

# нормальный текст
# [style flashback] [c eugene] флешбек и вещает Юджин

Блоки (ветвление)

// Вступительная сцена
# block intro
    ...
    [choice] Куда?|На работу->work;Домой->home;Остаться->stay

# block work
    ...
    -> home

# block home
    ...
    ///

# block stay
    ...
    -> intro
